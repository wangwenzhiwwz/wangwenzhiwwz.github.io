####下载Putty进行安装和vps连接

第一步：我们下载Putty进行安装和vps连接，我们可以选择百度云盘下载或者直接百度一下，资源很多，下载完成之后直接解压运行putty.exe即可：

百度云盘下载链接:https://pan.baidu.com/s/1S_oejt1EZ6iCdGVZCDzbMw

密码:9ix1

第二步

填入刚才查到的主机IP地址和端口号，点击打开！

第三步

如果这里有对话框弹出，选择是，然后在全黑的屏幕上输入 root ，回车。等五秒，按提示输入root的密码，密码就是前面刚刚重置的root密码。复制你的密码，粘贴至putty（粘贴方式为单击鼠标右键，只需要右键单击一次，这里不会显示任何内容，其实是已经输入了），回车。
