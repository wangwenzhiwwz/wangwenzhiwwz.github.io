    VPS 部署 ChatGPT

    今天使用的 VPS 来自 搬瓦工（机型：KVMV5-40G-2G-2T-CA-CN2GIA），系统为 Ubuntu 22.04 x86_64 。

    首先安装 Docker 服务，这里使用官方的一键脚本进行安装

    curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun

    Docker ChatGPT 部署命

    docker run \    
            --name chatgpt-web \    
            -p 3002:3002 \    
            --env OPENAI_API_KEY=Your-OpenAi-API \    
            --restart always \    
            -d chenzhaoyu94/chatgpt-web:latest

    部署完毕以后，访问 http://IP:3002 ，检测 ChatGPT 是否部署成功
