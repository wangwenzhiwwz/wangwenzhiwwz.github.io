sublime text3简体中文版汉化教程

今天发现sublime确实很方便，想弄个中文版的sublime，居然可以不用重新下载汉化包或者重新下载简体中文版了~而是只需要安装个插件即可！实在太方便了，牛逼烘烘的编辑器啊~大爱！！

1.启动并进入sublime主界面

2.安装汉化包：

使用快捷键：Ctrl+shift+p（或者：菜单 – Tools – Command Paletter），
输入package control: install package，

3.点击菜单栏中“preferences”，

弹出选项中找到“package control”；

4.进入下一步，输入“localization”

5.安装完之后

可以在菜单中查看到是否有此插件；

6.插件以及安装成功

现在我们就可以汉化啦，点击help菜单，找到“language”选择你想要的语言即可~~
