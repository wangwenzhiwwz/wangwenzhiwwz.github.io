共享Windows电脑文件

如何在局域网共享 Windows 电脑文件夹？

作者 王文志 日期 2018-07-23
网络
共享Windows电脑文件

在一个家庭或者办公室中，有几台甚至几十台电脑，一般情况下，他们都是同时连接在一个局域网内，如果能为特定的文件夹或者磁盘设置文件夹共享，我们就可以像在自己电脑中拷贝文件一样方便，而不需要使用 U 盘或者硬盘导出拷贝或者使用 QQ 等软件传来传去了。

以这台 Windows 10 系统为例：

首先，在右下角的网络图标上单击右键，选择【打开“网络和Internet ”设置】然后点击这里的【网络和共享中心】，在弹出的窗口中，点击【更改高级共享设置】，在来宾或公用网络这里，勾选【启用网络发现】、勾选【启用文件和打印机共享】，在所有网络这里，勾选【启用共享】、勾选【启用128位加密保护】，最后勾选【关闭密码保护共享】，最后保存更改。Windows7 系统更简单些，在任务栏的网络图标单击右键，点击【打开网络和共享中心】即可继续操作，其他设置与 Windows 10 几乎一样。

第二步，我们找到需要共享的文件夹或者盘符，以这台电脑为例，我们要共享的是整个 F 盘，它包含所有 Topbook 的文档，这样一来，其他后期大神可以直接在这里调取素材和包装进行制作，也可以把素材拷贝进来归档；运营小姐姐也可以从她的电脑上直接访问这里的成片然后在网络上分发上传。我们在盘符上单击右键，选择【属性】，或者像我们之前说过的，摁住 Alt 键的同时，双击盘符，直接调出属性。

在这里，我们先点击【安全】选项卡，这里，是使用这台电脑的所有用户和组，我们点击【编辑】，然后点击【添加】，在下方空白框内，输入 everyone，我们需要让任何人都有使用这个文件夹的权限。点击右侧的检查名称，当单词下有下划线的时候，直接点击确定，我们看到上方列表多了这个 Everyone 的账户，选中它，并且，把下方的【修改】、【读取和执行】两个选项后方的允许复选框勾选。然后点击【确定】。

之后，我们在这里点击【共享】选项卡，并且点击【高级共享】，勾选【共享此文件夹】，同时，点击下方的【权限】，勾选这里的【更改】和【读取】，点击【确定】，等待共享执行完成。依次点击【确定】。

到这里，共享就完成了。

如何访问？

我们在右下角点击网络图标，在所连接的网络上点击属性，下方的 IPv4地址，就是这台电脑的 IP 地址，我们记住它。

点击确定，如果让你输入凭据，这个凭据就是你正在使用的这台电脑的帐户名以及密码。最后即可找到我们刚刚共享的盘符。

你可以把它拖放到桌面成为快捷键方式，以便经常访问，也可以直接拖动图标到左侧栏的计算机图标上，会自动弹出映射驱动器对话框。它会帮你自动定义一个驱动器盘符，你也可以自己自定义，最后点击【完成】，今后每次打开【此电脑】，它会自动出现在【网络位置】，双击打开即可，就像访问自己的电脑一样方便。

Mac 系统如何访问刚刚的局域网共享文件呢？

打开 Finder 窗口之后，在键盘上使用 Command+K，弹出【访问服务器】对话框，在 smb:// 后面直接输入需要访问的电脑 IP 地址。你可以点击右侧 + 号，添加到个人收藏服务器，我们选择连接。再输入这台 Mac 的用户名和密码，稍等片刻，这里显示的就是共享的文件夹或者盘符，比如我们点击 F，再点击【好】，便成功访问。
