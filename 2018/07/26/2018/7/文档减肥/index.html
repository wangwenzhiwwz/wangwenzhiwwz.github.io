NXPowerLite Desktop 8.0.4

NXPowerLite Desktop 8.0.4 中文多语免费版。今天小编给大家分享一款文档减肥工具。在为文件过大而烦恼吗?文件过大难以通过电子邮件发送，并且浪费系统存储空间。XPowerLite Desktop 可大大减小过大的 Word、Excel、PowerPoint、JPEG 和 PDF 文件大小，使之可更为轻松地通过电子邮件进行发送。

NXPowerLite Desktop 在全球拥有一百多万用户，它可处理核心区文件冗余，使 PowerPoint、Word、Excel、JPEG 和 PDF 文件的压缩率高达 95%。它还可与您的电子邮件客户端集成，以在发送时自动压缩电子邮件附件。
对于 NXPowerLite Desktop 7， 我们的首要任务是整合、完善和细化其核心功能。因此, 基本这个主版本，我们已听取您的反馈意见，并根据您的需求进行了许多改进，包括：

更强的 PDF 优化功能
我们测试了数千个 PDF 文件，尽力寻求改善优化效果的途径。版本 5 的用户可能已经发现，它对有些文件的压缩效果很好，但对其它有此些文件效果不佳。对于版本 6，我们增加了一些重要的新优化功能，用户可以体验到两种文件格式显著提高的压缩一致性和效率。

全面支持 Office 2013 文件
NXPowerLite 仍可大幅降低文件大小，即使是最新版本的 Microsoft Office 也是如此。NXPowerLite 现在与 Office 2013 完全兼容，可以对 Office 2013 创建的文件进行优化。请注意，NXPowerLite 目前仅与 32 位版本的 Microsoft Office 应用程序兼容。

替换原文件或创建一个新的优化副本
出现最频繁的请求是选择替换原文件而非创建一个优化副本。在版本 8 中，将会以优化副本替换原文件作为默认操作。

NXPowerLite 和邮件压缩优化（仅适于 Outlook）
压缩功能对包含大量数据/文本的文件有良好的效果，而以图像为主的文档的优化效果最佳。我们已在您的邮件客户端将这两个解决方案合而为一。

浏览器中的单击选项
为使常规任务更快、更便捷，我们已在浏览器中添加了一系列右键单击选项。例如，如果您选择了五张照片，右键单击并选择“优化和电子邮件发送”选项，将会对五张图片进行优化并将其自动添加到新的电子邮件中。

百余项改进及优化
正如我们炫亮的新界面一样，我们在尽力改善用户对此款软件的整体体验。从简化基本工作流程到更清晰实用的消息提醒，NXPowerLite 已经过全面改版，希望给您带来愉悦的体验！

自动优化和压缩电子邮件附件（仅限 Outlook）
如果 PowerLite 在安装时已与 Outlook 集成，则会尝试自动优化 PDF、PowerPoint、Word、Excel 和 JPEG 附件。NXPowerLite 还具备压缩功能，确保您的附件尽量压缩到最小！

启用或禁用电子邮件附件优化（仅限 Outlook)
默认情况下, 在默认电子邮件客户端中打开的每一封新邮件都可以使用 NXPowerLite。

禁用 NXPowerLite
可以对某些电子邮件禁用 NXPowerLite。 若要执行此操作，只需在电子邮件撰写窗口中点击功能区按钮“NXPowerLite 已启用”即可。该按钮变为“NXPowerLite 已禁用”后， 在发送邮件之前 NXPowerLite 不会对附件进行优化或压缩。在您重新启用 NXPowerLite 之前，其将对所有电子邮件保持禁用状态。

如果我忘记禁用 NXPowerLite 会出现什么情况？
当您点击发送邮件时，NXPowerLite 会在其处理文件的同时显示一个进度对话框。在该屏幕上有两个功能：

“取消”将停止优化，并返回到撰写邮件窗口
“发送原始附件”按钮将停止优化，并发送含原始附件的电子邮件。

NXPowerLite 可以优化的文件选项
对于 PDF、PowerPoint、Word、Excel、JPEG 和 Zip 文件，在 Windows Explorer 中的右键点击文件可显示下列单击选项。

添加
打开 NXPowerLite 主屏幕，并添加准备优化的文件。如果 NXPowerLite 已打开, 文件将被添加到列表中。

优化
将立即优化文件；并基于您当前的“保存配置文件”， 替换原文件或在同一个文件夹中创建一个副本。

优化并压缩
将立即优化文件，并在原文件的压缩档中收集优化的版本。

优化并发送电子邮件
将立即优化该文件，并将其添加到在默认电子邮件客户端中新建的电子邮件。

优化、压缩并发送电子邮件
将立即优化和压缩文件，并将压缩档添加到在默认电子邮件客户端新建的电子邮件。

所有文件类型的选项
所有文件类型在 Windows Explorer 中都将有下列选项。

压缩
将立即压缩文件，并将压缩档存放在与原文件相同的文件夹中。

压缩并发送电子邮件
将立即压缩文件，并将压缩档添加到在默认电子邮件客户端中新建的电子邮件。

Windows XP SP3 或更高

NXPowerLite 文件支持
NXPowerLite 支持对以下文件的处理：

PDF
NXPowerLite 可以处理 1.4 到 1.7 版 8 级扩展的 PDF 文件，包括 Adobe Acrobat Pro 到 XI 版本创建的文件，同时也支持下列 PDF 子集：

PDF/A
PDF/X
PDF/E
PDF/UA
PDF/VT
note NXPowerLite 不优化 PDF 组合文件。
Microsoft Office
NXPowerLite 可以处理 Office 97 到 Office 2013 创建的有下列扩展名的 Microsoft Office 文件。将对以下扩展名的 Office 文件进行优化：

PowerPoint：pptx, pptm, ppt, ppsx, ppsm, pps, potx, potm, pot
Word：docx, docm, doc, dotx, dotm, dot
Excel：xlsx, xslm, xls, xltx, xltm, xlt
JPEG
NXPowerLite 可以对有以下扩展名的 JPEG 文件进行优化：

JPEG
JPG
JPE
JFIF
JIF
JFI
压缩文件
NXPowerLite 可以对含有上述任何格式的标准压缩文件进行优化
